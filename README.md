## Project Overview
This repository contains an interactive Python assignment exploring **Special Methods** (also known as **Dunder Methods**) in Object-Oriented Programming (OOP). 

Special methods are built-in Python methods identified by their double underscores (e.g., `__init__`). They allow custom-defined objects to interact seamlessly with Python's built-in operators and functions, making custom classes behave intuitively like native Python types.

## Concepts Covered
This assignment demonstrates the implementation of three foundational special methods using a custom `SmartPhone` class:

1. **`__init__` (The Initializer):** Automatically sets up the object's attributes (`brand`, `model`, and `storage_gb`) when a new instance is created.
2. **`__str__` (String Representation):** Configures a user-friendly, readable text output whenever the object is passed into `print()`.
3. **`__len__` (Length Definition):** Dictates what the built-in `len()` function returns when called on the object (mapped to the phone's storage capacity in this example).
